# KAYZ CareLink

**An advanced, AI-powered remote patient monitoring platform designed for modern clinical environments and care homes.**

KAYZ CareLink bridges the gap between continuous biometric hardware tracking and actionable clinical insights. By seamlessly integrating data from wearable sensors and room-based gateway hubs, this platform provides healthcare professionals with a unified, real-time dashboard to monitor patient vitals, manage hardware assignments, and respond instantly to critical health alerts.

![KAYZ CareLink Dashboard](https://images.unsplash.com/photo-1551076805-e1869033e561?auto=format&fit=crop&q=80&w=1200)

## 🚀 Key Features

### 🧠 AI Clinical Trend Snapshot
Say goodbye to manual data crunching. Our built-in AI engine automatically analyzes 3, 7, 14, or 30 days of raw biometric telemetry (Blood Pressure, Heart Rate, SpO2, and Glucose). It computes percentage variances, checks against patient-specific thresholds, and outputs a highly readable, actionable clinical summary in under a second.

### 🚨 Intelligent Alert & Triage System
- **Global Alert Banner:** A highly visible, non-intrusive banner that alerts staff immediately to critical threshold breaches across the facility.
- **Smart Notifications:** A dropdown feed that aggregates unresolved warnings and critical alerts.
- **Instant Resolution:** One-click navigation to the affected patient's profile to review AI insights, adjust medications, or verify sensor placement. 

### 🛡️ Compliance & Audit Logging
A permanent, immutable record of every alert generated by the system. Every acknowledged, dismissed, or unresolved alert is tracked with strict timestamps and operator states to ensure compliance with healthcare auditing standards. **Records cannot be deleted.**

### 💻 Complete Hardware Management
Manage the physical layer of your clinical environment directly from the UI.
- **Sensor Assignment:** Pair continuous monitoring wearables directly to specific patients.
- **Gateway Hubs:** Register and track the status of network hubs deployed across facility rooms to ensure 100% telemetry uptime.

### 📊 Dynamic Patient Profiles
Interactive charting powered by Recharts allows clinicians to scrub through historical biometric data effortlessly. Every patient profile supports custom, individual safety thresholds (e.g., custom Max Systolic BP), ensuring that alerts are tailored to the patient's specific baseline.

## 🛠️ Tech Stack

- **Frontend Framework:** React 18 with TypeScript
- **Styling & UI:** Tailwind CSS, Framer Motion (Animations), Lucide React (Icons)
- **Data Visualization:** Recharts
- **State Management:** React Context API
- **Backend / Database:** Supabase (PostgreSQL)

## 📦 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/kayz-carelink.git
   cd kayz-carelink
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Start the Development Server**
   ```bash
   npm run dev
   ```

## 🔒 Security & Compliance
This platform is designed with healthcare data abstraction in mind. Patient identifiable data should be handled strictly according to HIPAA / GDPR regulations. All alerts and biometric transmissions require secure, authenticated database connections via Supabase Row Level Security (RLS).
